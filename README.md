# Path-Loss-ML-Optimization

# Predicción de Path Loss mediante Modelos de Machine Learning 
# (Path Loss Prediction using Machine Learning Models)

## 🇪🇸 Resumen del Proyecto 

Este repositorio se enfoca en la implementación y evaluación de algoritmos de aprendizaje supervisado para predecir la pérdida de trayectoria en redes inalámbricas. Se priorizó el uso de modelos robustos capaces de manejar la no linealidad de las señales de telecomunicaciones.

### Modelos Implementados:
* **XGBoost (Extreme Gradient Boosting):** Utilizado por su alta eficiencia y precisión en el manejo de datos tabulares complejos provenientes de mediciones de RSSI.
* **KNN (K-Nearest Neighbors):** Implementado como modelo de referencia para evaluar la relación espacial y la proximidad de los datos de señal.
* **Procesamiento:** Incluye la limpieza de datos, manejo de valores atípicos (outliers) y normalización para asegurar la convergencia de los modelos.

---

## 🇺🇸 Project Overview 

This repository focuses on implementing and evaluating supervised learning algorithms to predict path loss in wireless networks. The project prioritizes robust models capable of handling the non-linear nature of telecommunications signals.

### Implemented Models:
* **XGBoost (Extreme Gradient Boosting):** Leveraged for its high efficiency and accuracy in handling complex tabular data from RSSI measurements.
* **KNN (K-Nearest Neighbors):** Implemented as a baseline model to evaluate spatial relationships and signal data proximity.
* **Data Processing:** Includes data cleaning, outlier handling, and normalization to ensure model convergence and reliability.
